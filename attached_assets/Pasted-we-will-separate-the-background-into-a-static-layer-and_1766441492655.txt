we will separate the background into a static layer and the UI into an animated "floating" layer. This ensures that the background never moves, even if the user interacts with the buttons or triggers an overscroll bounce.

1. Global Setup (The Foundation)

You must ensure these two files are updated to prevent the "white gap" and allow the app to use the full screen.

index.html Add viewport-fit=cover to your meta tag:

HTML
<meta name="viewport" content="width=device-width, initial-scale=1.0, viewport-fit=cover" />
index.css (Global Styles) This locks the root container so the "rubber band" effect doesn't show the browser background:

CSS
html, body {
  background-color: #000000;
  margin: 0;
  padding: 0;
  height: 100%;
  width: 100%;
  overflow: hidden;
  position: fixed; /* Prevents the document from bouncing */
}

@keyframes fadeInUp {
  from {
    opacity: 0;
    transform: translateY(20px);
  }
  to {
    opacity: 1;
    transform: translateY(0);
  }
}

.animate-fade-in {
  animation: fadeInUp 0.8s ease-out forwards;
}
2. The Integrated Landing Component

This implementation uses the "Dual-Layer" approach with the requested fade-in animation.

TypeScript
import { Button } from "@/components/ui/button";
import { useLocation } from "wouter";
import backgroundImage from "@assets/landing-background.png";
import logo from "@assets/logo2_1762477206651.png";

export default function Landing() {
  const [, setLocation] = useLocation();

  return (
    <div style={{ width: '100%', height: '100%', backgroundColor: '#000000' }}>
      
      {/* LAYER 1: The Static Background (Never Moves) */}
      <div 
        style={{
          position: 'fixed',
          top: 0,
          left: 0,
          right: 0,
          bottom: 0,
          zIndex: 0,
          backgroundImage: `url(${backgroundImage})`,
          backgroundSize: 'cover',
          backgroundPosition: 'center',
          backgroundRepeat: 'no-repeat',
        }}
      />

      {/* LAYER 2: The Content (Detached & Animated) */}
      <div 
        className="animate-fade-in"
        style={{
          position: 'relative',
          zIndex: 1,
          width: '100%',
          height: '100vh',
          display: 'grid',
          gridTemplateRows: '1fr auto 1fr auto',
          alignItems: 'center',
          justifyItems: 'center',
          // Safe area ensures content doesn't hit notch/home bar
          paddingTop: 'env(safe-area-inset-top)',
          paddingBottom: 'env(safe-area-inset-bottom)',
        }}
      >
        {/* Row 1: Spacer */}
        <div />

        {/* Row 2: Logo */}
        <img 
          src={logo} 
          alt="BoxStat Logo" 
          style={{
            width: '180px',
            height: 'auto',
            filter: 'drop-shadow(0 10px 20px rgba(0,0,0,0.5))',
          }}
        />

        {/* Row 3: Spacer */}
        <div />

        {/* Row 4: Bottom CTA */}
        <div 
          style={{
            display: 'flex',
            flexDirection: 'column',
            alignItems: 'center',
            gap: '16px',
            // max() ensures a minimum of 60px padding OR the safe area height
            paddingBottom: 'max(60px, env(safe-area-inset-bottom))', 
            width: '100%',
          }}
        >
          <Button 
            size="lg" 
            onClick={() => setLocation('/registration')}
            style={{
              backgroundColor: 'rgba(0,0,0,0.2)',
              color: 'white',
              fontWeight: 'bold',
              padding: '24px 48px',
              borderRadius: '12px',
              border: '1px solid rgba(255,255,255,0.1)',
              backdropFilter: 'blur(8px)',
              minWidth: '280px',
              fontSize: '14px',
              letterSpacing: '0.1em',
            }}
          >
            LET'S GO
          </Button>

          <p style={{ color: 'white', fontSize: '14px', margin: 0 }}>
            <span style={{ opacity: 0.8 }}>HAVE AN ACCOUNT? </span>
            <button 
              onClick={() => setLocation('/login')}
              style={{
                color: 'white',
                fontWeight: 'bold',
                background: 'none',
                border: 'none',
                marginLeft: '4px',
                fontSize: '14px',
              }}
            >
              SIGN IN
            </button>
          </p>
        </div>
      </div>
    </div>
  );
}
Why this works for iOS:

No Gaps: viewport-fit=cover tells iOS to fill the screen, and the position: fixed background layer acts as a permanent canvas.

Home Bar/Notch Protection: env(safe-area-inset-bottom) pushes the "Sign In" text up just enough so the iPhone's home swipe bar doesn't cover it.

Visual Polish: The animate-fade-in class (defined in CSS) makes the UI slide up and fade in over the static background, giving it a premium app feel.

Rubber-band Protection: Because the background is in its own fixed layer (Layer 1), even if the content (Layer 2) "bounces," the image stays perfectly still behind it.
